# ComputerVisionFinal
This program analyzes photos of weeds and classifies them.

## Introduction
The intended purpose of this program is to take a structured image file dataset, load it, and train a model to predict the type of weed photographed.

## How to use
To test/run this program you can use a.) GitHub, to run each cell, b.) Google Colab, where it was originally created, c.) Jupyter Notebook via browser, or d.) any other method you desire.
